# Repair System Tracking Backend API

A comprehensive backend platform designed to streamline operations for repair shops.

This application provides the API services for managing client appointments, tracking repair statuses, generating automated invoices, and facilitating seamless real-time communication through WebSockets and Google Gemini AI integration.

## Features
- **Repair Tracking API**: Real-time status updates from ticket creation to completion.
- **Live Support Chat (WebSockets)**: Instant bidirectional messaging service.
- **AI Chatbot Service**: Automated initial triaging using Gemini AI.
- **Invoice Generation**: Automated PDF invoice generation.
- **Notification System**: In-app and email alerting system.

## Tech Stack
- **Runtime**: Node.js
- **Framework**: Express.js
- **Real-time**: Socket.io
- **Database**: MongoDB (Mongoose)
- **Authentication**: JWT, bcrypt
- **AI**: Google Gemini Pro
